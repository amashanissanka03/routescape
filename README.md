# RouteScape

RouteScape is a tourist planning web application developed using PHP, MySQL, Bootstrap, and Leaflet.

## Features

### Tourist User
- View list of attractions
- Filter attractions by category
- View detailed information (description, opening times, travel tips, distance)
- View attraction locations on map
- Create a one-day trip plan

### Administrator
- Add new attractions
- Edit attraction details
- Delete attractions

## Technologies Used
- PHP
- MySQL
- Bootstrap
- Leaflet (for maps)

## How to Run
1. Place the project folder in Laragon's `www` directory
2. Start Laragon services
3. Import the database (`routescape_db`)
4. Open in browser:
   http://localhost/routescape
